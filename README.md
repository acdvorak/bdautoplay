# Blu-ray AutoPlay

## Supported OSes

*  Windows 7 x86 (32-bit) and x64 (64-bit)

## Supported Media Players

*  Media Player Classic - Home Cinema (32- and 64-bit)

## Building from Source

Requires Visual Studio 2010 or newer.
